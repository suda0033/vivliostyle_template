# 管理者ログイン

## 機能概要

管理者ログイン機能は、運用担当者が処理状況、取込結果、エラー明細を確認するための管理画面へアクセスする機能である。

## 入力項目

| 項目 | 必須 | 型 | 桁数 | 備考 |
| --- | --- | --- | --- | --- |
| ログイン ID | 必須 | 文字列 | 64 | メールアドレス形式 |
| パスワード | 必須 | 文字列 | 128 | 画面上はマスク表示 |

## 処理仕様

### 認証処理

1. ログイン ID とパスワードの未入力チェックを行う。
2. 管理者ユーザーテーブルからログイン ID に一致する有効ユーザーを取得する。
3. パスワードハッシュを照合する。
4. 認証成功時はセッションを発行し、処理状況一覧へ遷移する。
5. 認証失敗時はエラーメッセージを表示する。

### アカウントロック

| 条件 | 動作 |
| --- | --- |
| 連続 5 回失敗 | アカウントを一時ロックする |
| ロックから 30 分経過 | 自動解除する |
| 管理者による解除 | 即時解除する |

## アクセスするデータ

| データ名 | アクセス | 用途 |
| --- | --- | --- |
| 管理者ユーザーテーブル | 参照 | 認証、権限取得 |
| ログイン失敗履歴 | 登録、更新 | 失敗回数とロック状態の管理 |
| 操作ログ | 登録 | ログイン成功、失敗の監査 |
